THƯ MỤC VIDEO SẢN PHẨM

Bạn copy video demo sản phẩm vào đây, ví dụ:
- roboball-demo.mp4
- tay-cam-demo.mp4

Sau đó mở file assets/site.json và điền đường dẫn:
"videos": ["assets/videos/roboball-demo.mp4"]

Có thể dùng link YouTube thay cho file mp4, ví dụ:
"videos": ["https://www.youtube.com/watch?v=VIDEO_ID"]
